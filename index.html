Preguntas y respuestas

Estas son algunas curiocidades sobre mi persona respecto a mi enfoque como programadora.

¿Cuanto sabes de frontend?

Tengo un conocimiento bastante extenso sobre desarrollo frontend HTML, CSS, y JavaScript puro, aunque tambien conozco algunos frameworks populares como react, angular y vue, espero seguir mejorando en el frontend, suelo hacer cursos en mi tiempo libre para coseguir mas practia y dominarlo por completo.

¿En que tipo de proyectos haz trabajado?

Junto con unos compañeros realizamos un repositorio de github donde simulamos la interfaz de la pagina web de un banco, tambien desarrolamos paginas de tiendas, algunos juegos como practica utilizando HTML, CSS y JS y yo he realizado un programa para gestionar citas entre otros datos de un hospital con Java.

¿Cual es tu mision y vision?

Como programadora, mi misión es crear soluciones tecnológicas innovadoras y significativas que impacten positivamente la vida de las personas. Me comprometo a desarrollar software eficiente y sostenible, utilizando las últimas tecnologías y mejores prácticas de programación. Mi enfoque incluye colaborar con equipos diversos y Visualizo un futuro donde las soluciones que desarrollo no solo resuelven problemas, sino que también inspiran y mejoran la calidad de vida de las personas. Mi visión abarca la creación de software que sea accesible para todos, que promueva la inclusividad y que se adhiera a estándares éticos elevados.
